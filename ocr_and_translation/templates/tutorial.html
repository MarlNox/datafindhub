First Step: Set Up
1. Create a Folder in your Google Drive named "full_screenshots".
Second Step: File Preparation
1. Download Company data from your portal of choice.
2. Create a file containing Company Name, BvD Number and Website Address as per the provided template.
3. Save File in .txt or .csv file format.
Third Step: Conduct the Scraping
1. Upload the .csv or .txt file containin the Extracted Websites, Names and BvD Numbers of the companies being researched in the main page.
2. Assign a name to the output file which will contain the extracted information.
3. Wait for the process to finish creating screenshots, extracting them and providing the final analytical table.
4. Use the highlighter option to highlight positively correlated keywords, and negatively correlated keywords with your search/analysis.
5. All the captured screenshots, along with the finalized table are contained in your Google Drive folder named full_screenshots, folderized based on company names.
